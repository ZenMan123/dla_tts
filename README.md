# DLA TTS Task

Все шаги для воспроизведения HiFiGAN описаны в ноутбуке [demo.ipynb](demo.ipynb)

Ран, в котором обучается финальная моделька - [тык](https://www.comet.com/artemsafin67/dla-tts/bwv4h6vym8c0jneptdrauywi9aklj5sn?compareXAxis=step&experiment-tab=panels&showOutliers=true&smoothing=0&viewId=new&xAxis=step)

Обучал классический HiFiGAN по мотивам оригинальной статьи. Во время обучения использовались gt мел-спектрограммы. Во время инференса для получения мел-спектрограмм использовалась модель [nvidia/tts_en_fastpitch](https://huggingface.co/nvidia/tts_en_fastpitch)

Заметил, что распределение предсказанных мел-спектрограмм, полученных из модели сильно отличается от gt мел-спектрограмм. В результате сгенерированные аудио получались тихими и очень шумными.
Проблему решил путем масштабирования предсказанных мел-спектрограмм (сдвинул их на определенную константу, выровнял среднее).

Более подробный анализ модели можно посмотреть в [репорте](report.ipynb). Для его запуска достаточно установить все зависимости из [requirements.txt](requirements.txt)
